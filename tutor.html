課程宗旨：引領未來，對話科技

本課程旨在培養對 硬體實作 與 AI 輔助編程 (Vibe Coding) 教學有熱忱之講師。我們將賦予您引導學員透過 自然語言指令 驅動程式碼生成的能力。

課程設計循序漸進，從基礎電子概念到複雜專案整合，並全面融入 Vibe Coding 的應用，讓程式設計如同對話般直覺。您將具備引導學生進行科技探索、培養創新思維及解決問題的專業素養。

  • 掌握 AI 編程： 利用 Vibe Coding 簡化程式設計，專注於創意實現。
  • 深入硬體實作： 從基礎電子到複雜專案的完整學習路徑。
  • 培養教學專業： 學習有效引導學員探索科技與解決問題。

課程內容：從基礎到專家，全面培養

第一階段：Arduino 硬體與 Vibe Coding 入門 (3 堂)

透過 Arduino 實作，引導學員認識基本電學概念、電子元件，學習數位與類比訊號的輸入與輸出，並以 Vibe Coding 為核心，學會如何用直覺的中文指令控制硬體。

第 1 堂：Arduino 簡介與 Vibe Coding 初探：點亮你的第一盞燈

  • 概念講解：Arduino 簡介、Vibe Coding 角色、開發環境準備、電路基礎。
  • 實作練習：Vibe Coding 初體驗，點亮板載 LED 與外部 LED。

第 2 堂：Vibe Coding 與多個 LED 控制：打造你的燈光秀

  • 概念講解：數位輸出延伸、時間序列、Vibe Coding 邏輯處理能力。
  • 實作練習：多個 LED 接線、Vibe Coding 控制多個 LED、紅綠燈模擬。

第 3 堂：Vibe Coding 與數位輸入：按鈕控制

  • 概念講解：數位輸入、按鈕原理、下拉電阻的重要性。
  • 實作練習：連接按鈕與下拉電阻、Vibe Coding 讀取按鈕狀態與控制 LED。

第二階段：感測器與執行器：讓 Arduino 感知與行動 (4 堂)

學習不同類型的感測器 (類比輸入) 和執行器，並透過 Vibe Coding 實現 Arduino 對環境的感知和反應，讓專案更加智慧化。

第 4 堂：Vibe Coding 與類比輸入：可變電阻

  • 概念講解：類比輸入、可變電阻、PWM 概念。
  • 實作練習：連接可變電阻、讀取類比值、控制 LED 亮度。

第 5 堂：Vibe Coding 與光線感測：光敏電阻

  • 概念講解：光敏電阻、分壓電路。
  • 實作練習：連接光敏電阻、讀取光線值、實現自動夜燈。

第 6 堂：Vibe Coding 與聲音反應：蜂鳴器

  • 概念講解：蜂鳴器原理、聲音輸出、聲音頻率控制。
  • 實作練習：連接蜂鳴器、播放簡單音效、模擬警報聲。

第 7 堂：Vibe Coding 與運動控制：伺服馬達

  • 概念講解：伺服馬達、PWM 訊號控制。
  • 實作練習：連接伺服馬達、控制伺服馬達轉動、結合可變電阻控制角度。

第三階段：進階感測與資料處理：讓 Arduino 更聰明 (4 堂)

學習更複雜的感測器應用，如溫濕度、距離感測，並透過 Vibe Coding 處理感測資料，實現更精確的環境感知與智能反應。

第 8 堂：Vibe Coding 與溫濕度感測：DHT11

  • 概念講解：DHT11 原理、數位通訊。
  • 實作練習：連接 DHT11、讀取溫濕度、實現簡易環境監測與警示。

第 9 堂：Vibe Coding 與距離感測：超音波模組

  • 概念講解：HC-SR04 原理、聲速計算。
  • 實作練習：連接超音波模組、讀取距離、實現簡易防盜警報。

第 10 堂：Vibe Coding 與 RGB LED：色彩變幻

  • 概念講解：RGB LED 原理、PWM 與混色。
  • 實作練習：連接 RGB LED、控制單一顏色、實現多種顏色切換與呼吸燈效果。

第 11 堂：Vibe Coding 與馬達控制：直流馬達

  • 概念講解：直流馬達原理、L298N 馬達驅動模組。
  • 實作練習：連接直流馬達與 L298N、控制馬達轉動、結合按鈕控制馬達。

第四階段：物聯網基礎與進階互動：讓 Arduino 互聯互通 (4 堂)

引入物聯網概念，學習藍牙通訊，並透過 Vibe Coding 實現 Arduino 與手機的無線互動，理解智慧裝置的運作原理。

第 12 堂：Vibe Coding 與 OLED 顯示器：文字與圖形輸出

  • 概念講解：OLED 顯示器、I2C 通訊協議。
  • 實作練習：連接 OLED 顯示器、顯示文字與感測器數據。

第 13 堂：Vibe Coding 與藍牙模組通訊：基礎連結

  • 概念講解：藍牙通訊基本概念、HC-05/06 模組、序列埠通訊。
  • 實作練習：連接藍牙模組、藍牙配對與測試、藍牙傳送接收數據。

第 14 堂：Vibe Coding 實現手機遙控 Arduino：遠端控制

  • 概念講解：指令解析、狀態判斷與控制。
  • 實作練習：藍牙遙控燈、藍牙遙控風扇。

第 15 堂：Vibe Coding 與專案整合：智能安全門

  • 概念講解：專案整合挑戰、Vibe Coding 進階邏輯串聯。
  • 實作練習：智能安全門設計與逐步實現、測試與除錯。

第五階段：進階感測與資料處理：讓 Arduino 更聰明 (4 堂)

學習更複雜的感測器應用，如溫濕度、距離感測，並透過 Vibe Coding 處理感測資料，實現更精確的環境感知與智能反應。

第 16 堂：Vibe Coding 與網路通訊：ESP32 Wi-Fi 基礎

  • 概念講解：物聯網 (IoT)、ESP32 簡介、Wi-Fi 連線基礎。
  • 實作練習：ESP32 環境建置、連接 Wi-Fi、透過 Wi-Fi 控制 LED。

第 17 堂：Vibe Coding 與網路數據發送：ThingSpeak 雲端平台

  • 概念講解：雲端數據平台、API Key 與數據上傳。
  • 實作練習：ThingSpeak 設定、上傳溫濕度數據到 ThingSpeak。

第 18 堂：Vibe Coding 與智慧家庭整合：簡易智能燈

  • 概念講解：智慧家庭概念、MQTT 協議 (可選)、Vibe Coding 整合多重條件。
  • 實作練習：整合多個感測器與執行器、實現智能夜燈。

第 19 堂：專案衝刺與 Vibe Coding 輔助除錯

  • 概念講解：專案整合挑戰、模組化思維、Vibe Coding 進階應用、除錯技巧。
  • 實作練習：學員自選專案實作、Vibe Coding 輔助除錯。

第六階段：專案整合與教學實踐：成為智能創客導師 (2 堂)

綜合運用所學知識，完成綜合專案，並學習如何有效地輔導他人使用 Vibe Coding 進行創作，培養成為智能創客導師的專業素養。

第 20 堂：教學法與成果發表

  • 概念講解：智能創客教學法、Vibe Coding 在教學中的應用、問題解決策略。
  • 實作練習：專案成果發表、教學模擬與回饋、未來展望。
